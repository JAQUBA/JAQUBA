## About

Poland-based electronics technician and developer focused on embedded systems, native desktop tools, and real hardware workflows.

- STM32, AVR, ESP32-H2, C6, S3
- Native C and C++ tooling for hardware communication and diagnostics
- Custom frameworks, telemetry systems, and protocol-heavy projects
- Electronics repair, diagnostics, and reverse engineering
